Key Projects & Achievements

  • File Compression System using Huffman Coding -Designed and implemented a file compression utility leveraging Huffman Coding, a greedy algorithm that minimizes file size by assigning variable-length binary codes based on character frequencies. This project involved building a frequency table, constructing a Huffman Tree, generating binary codes, and enabling both compression and decompression of text files. It demonstrates effective use of data structures like priority queues and binary trees to optimize storage efficiency.
